# DigitecScript

## Installation
To run the script you need to install the following packages:
- selenium==4.9.1
- pandas==2.0.1
- bs4==0.0.1
- requests==2.30.0

you can install them with the following command:
```
pip install -r requirements.txt
```
## Usage
To run the script you need to run the following command:
```
python main.py
```
